# StreetComb

Mobile-first fighting game for Android where the player performs movement, attacks, and specials through finger gestures drawn directly on the screen.

## Core idea
- Touch-native combat
- Gesture-based moves and combos
- Fast, readable, stylish fights
- Distinct fighters with different combat grammar
- Tournament / arcade progression with light roguelite structure

## Current goal
Build a vertical slice that proves gesture fighting feels good on mobile.

## Vertical slice targets
- 2 playable fighters: Raze and Iron Monk
- 1 arena
- 1 short tournament run
- 6-8 reliable gestures
- Basic HUD
- Enemy AI
- Android test build pipeline later
